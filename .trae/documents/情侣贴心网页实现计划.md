# 💕 情侣贴心网页 — 实现计划

## 摘要

为情侣打造的贴心网页应用，包含 8 个实用功能模块：在一起天数计算器、饮食热量计算器、天气查询、月经周期记录、待办事项、课程表导入、心情记录、加密小秘密。采用纯 HTML/CSS/JS 单文件方案，数据通过 localStorage 持久化，设计风格追求浪漫温馨的手作质感。

---

## 当前状态分析

- **起点**：从零开始，无现有代码
- **技术选型**：纯 HTML/CSS/JS，无框架、无构建工具
- **部署方式**：单文件，双击即可打开，方便分享给对方
- **数据存储**：浏览器 localStorage

---

## 技术调研结论

### 免费天气 API
- **首选**：Open-Meteo（完全免费、无需 API Key、支持中国城市）
  - Geocoding API：`https://geocoding-api.open-meteo.com/v1/search?name=城市名&language=zh`
  - Forecast API：`https://api.open-meteo.com/v1/forecast?latitude=...&longitude=...`
- **备选**：wttr.in（`https://wttr.in/城市名?format=j1`），作为 fallback

### 中文字体（Google Fonts）
- **标题/装饰**：ZCOOL XiaoWei（站酷小薇体，文艺温馨）
- **正文/UI**：Noto Serif SC（思源宋体，优雅清晰）
- **特殊强调**：Ma Shan Zheng（马善政楷体，手写感）
- **国内备选镜像**：fonts.loli.net

### 加密方案
- Web Crypto API + AES-GCM + PBKDF2（100,000 次迭代）
- 密码不存储，仅存储 salt 和验证 token

## 设计方案

### 色彩系统
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色（玫瑰色） | 温暖玫瑰 | `#D4756B` |
| 浅玫瑰 | 浅玫瑰 | `#E8A598` |
| 背景色 | 奶油白 | `#FDF6EC` |
| 强调色 | 暖金色 | `#C9A96E` |
| 主文字 | 深棕 | `#3D2C2C` |
| 次要文字 | 中棕 | `#7A6363` |

### 设计风格
- **追求**：手写信的温度感、复古纸张质感、不对称但和谐的布局
- **避免**：渐变紫色背景、大面积毛玻璃、过度圆角、AI 通用审美
- **装饰**：CSS 纸张纹理、心形跳动动画、手绘风分隔线

### 布局架构
- 单页应用（SPA），底部 4 个 Tab 导航
- Tab 分组：首页（天数+天气）| 记录（饮食+月经+心情）| 课程 | 更多（待办+秘密+设置）
- 响应式：手机（默认）→ 平板（768px+）→ 桌面（1024px+）

## 8 个功能模块实现方案

### 1. 在一起天数计算器（首页顶部）
- 大号数字显示天数 + "年 X 月 X 天"分解
- 可编辑纪念日日期
- 显示下一个纪念日倒计时

### 2. 饮食热量计算器（记录 Tab）
- 今日总热量/目标热量进度条
- 按餐别分组（早/午/晚/零食）
- 内置 ~30 种常见食物热量参考表
- 添加食物表单

### 3. 天气查询（首页）
- 当前温度、天气描述、体感温度、湿度、风速
- 未来 3 天预报（横向卡片）
- Open-Meteo API + 30 分钟缓存 + wttr.in fallback

### 4. 月经周期记录（记录 Tab）
- 日历视图标记经期日期
- 状态提示：经期第 X 天 / 距下次 X 天 / 排卵期
- 基于最近 3 次周期平均值预测
- 症状标签记录

### 5. 待办事项（更多 Tab）
- 快速添加 + 分类筛选（约会/生活/学习/工作）
- 优先级标记（高/中/低）
- 勾选完成（划线）+ 删除
- 今日/本周完成统计

### 6. 课程表（课程 Tab）
- 周视图表格（周一到周日，1-12 节）
- 添加/编辑课程弹窗
- JSON 格式导入（文本框粘贴）
- 当前周高亮 + 时间指示线

### 7. 心情记录（记录 Tab）
- 5 级心情选择器 + 日期 + 文字 + 标签
- 心情日历热力图
- 月度心情统计

### 8. 小秘密（更多 Tab）
- 首次设置密码（两次确认）
- AES-GCM 加密存储
- 查看需再次输入密码
- 密码修改功能

## 假设与决策

| 决策 | 理由 |
|------|------|
| 采用单文件方案 | 部署最简，方便情侣间分享 |
| 使用 Open-Meteo 天气 API | 免费、无需 Key、中国城市覆盖好 |
| 使用 Web Crypto API 加密 | 浏览器原生支持，无需第三方库 |
| localStorage 持久化 | 无需服务器，纯前端方案 |
| Google Fonts 中文字体 | 提升视觉品质，提供国内镜像备选 |

## 验证步骤

1. **功能验证**：逐一测试 8 个功能模块的 CRUD 操作
2. **数据持久化**：刷新页面后数据不丢失
3. **响应式**：在手机（375px）、平板（768px）、桌面（1024px）下测试布局
4. **天气 API**：测试正常返回和 fallback 场景
5. **加密安全**：验证密码错误时无法解密，密码正确时可正常加解密
6. **性能**：中文字体加载不阻塞渲染，页面切换流畅
7. **浏览器兼容**：Chrome、Firefox、Safari、Edge 主流浏览器测试
